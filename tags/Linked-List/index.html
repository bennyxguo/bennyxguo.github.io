本期讲说最常见的数据结构类型分别有数组、链表、跳表。这一期我们一起来了解它们的原理与实现。

  • Python: list = []
  • JavaScript: let x = [1, 2, 3]

当今的高级数据语言中，对于数组里面的类型没有严格要求，相对来说比较多元化。
